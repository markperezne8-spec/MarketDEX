# RC Delivery Operator Flow

`🟢 merged source -> ⬇️ pull when instructed -> 📦 manual delivery build -> 📄 download operator package -> 📁 extract outside repository -> 🚀 launch MarketDEX.exe -> 👁️ confirm first window`

Each signal represents a separate operator concept. Pulling source does not download the executable. Downloading the executable does not modify source authority.
