# Operator Flow — Direct Workspace Access

1. Open MarketDEX.
2. Click Inventory, Pricing, or Listing Workflow directly to review that workspace.
3. Select an inventory asset when performing asset-specific work.
4. Use Continue to Pricing for the guided pricing handoff.
5. Review pricing and use Continue to Listing Workflow for the guided listing handoff.
6. Clearing the selection disables guided handoff actions but does not eject the operator from the workspace being reviewed.
