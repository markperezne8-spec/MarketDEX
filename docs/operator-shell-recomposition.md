# Operator Shell & Workspace Recomposition

MarketDEX keeps its existing SQLite authority, repositories, services, pricing logic, listing workflow, and sale completion boundaries.

The desktop shell now presents those capabilities as operator workspaces:

- **Mission Control** — business snapshot only.
- **Inventory & Pricing** — inventory intake, management, pricing, and profit decisions.
- **Listings** — listing plan, preparation, review, handoff, marketplace outcome, and sale completion.

This is a presentation-boundary recomposition. It does not create a second data path or duplicate business authority.
