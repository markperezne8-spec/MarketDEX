# Direct Workspace Tab Access Test Matrix

| Scenario | Expected |
| --- | --- |
| App opens with no asset selected | Inventory, Pricing, and Listing Workflow tabs are clickable |
| No asset selected | Guided Continue buttons are disabled |
| Asset selected | Guided Continue buttons are enabled |
| Continue to Pricing clicked | Pricing workspace opens |
| Continue to Listing Workflow clicked | Listing Workflow opens |
| Selection cleared while Listing Workflow is open | Listing Workflow remains open and directly reviewable |
