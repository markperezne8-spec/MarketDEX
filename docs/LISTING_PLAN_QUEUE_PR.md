# PR: Add Listing Plan Queue operator workflow

Turns persisted Listing Decision Workspace plans into an offline-first operator queue. Saved plans appear with asset, marketplace, target price, and target ROI. Open Selected Plan returns to the authoritative inventory asset where the existing saved-plan restore flow repopulates the workspace.

Includes focused offscreen Qt operator-flow coverage, integration contract coverage, two CI gates, operator documentation, and the restored MarketDEX build progress tracker.

No marketplace publication occurs in this build.
