# MarketDEX Engineering Council

## Purpose
Every major milestone is reviewed from six perspectives.

1. 👤 Product Owner
2. 🧠 Chief Software Architect
3. 🤖 Senior Software Engineer
4. 🔍 Quality Assurance Lead
5. 📐 Systems Strategist & Product Evolution Lead
6. 🎨 User Experience Lead

## Approval Gate
A milestone is complete only when:
- Acceptance criteria are met
- Documentation is updated
- Tests pass
- Architecture remains consistent
- UX is reviewed
- Technical debt is recorded
