# Inventory Pricing Decision Workflow

MarketDEX pricing decision support now follows one connected operator flow:

1. Select one active inventory asset.
2. Enter marketplace, fee, shipping, and packaging costs.
3. Choose a target ROI percentage.
4. Review break-even, minimum profitable, and recommended sale prices.
5. Use **Use Recommended Price** to copy the current recommendation into Target Sale Price.
6. Review Sale Readiness and True Profit/ROI before listing.

The recommendation is recalculated from authoritative inventory unit cost plus selling costs. The button updates decision input only; it does not create a sale, listing, or authority event.
