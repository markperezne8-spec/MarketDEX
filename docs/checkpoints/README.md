# MarketDEX Checkpoint Addenda

This directory holds compact, repository-preserved checkpoint addenda when a safe full-document maintenance pass for the large living checkpoint authorities is not available.

- `FoundationCheckpoint.md` remains the current-state and resume authority.
- `CheckpointManifest.md` remains the permanent historical index.
- Addenda must not silently redefine either authority.
- Each addendum must identify the completed sequence, verified CI, merge commits, explicit non-goals, visual status, and exact next gate.
