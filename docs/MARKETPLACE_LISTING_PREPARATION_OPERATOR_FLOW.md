# Marketplace Listing Preparation Operator Flow

1. Select an inventory asset.
2. Complete pricing and save a listing plan.
3. Listing Execution Readiness validates the saved plan.
4. A passing asset becomes READY TO PREPARE.
5. MarketDEX displays the offline listing package.
6. The operator reviews the marketplace, title, quantity, target price, and asset ID.

The flow stops before marketplace publication.
