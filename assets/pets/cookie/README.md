# Cookie — ChatGPT Work Pet

Cookie is a custom electric collector companion created for ChatGPT Work.

## Repository package

Download `Cookie_Work_Pet_Repository_Ready.zip` from the ChatGPT deliverable, extract it, and copy the `Cookie_Work_Pet` folder here at:

`assets/pets/cookie/`

The package contains the validated 1536×1872 transparent sprite sheet, all nine animation previews, an all-states GIF and MP4, and validation output.

## Production asset

`assets/cookie_sprite_sheet.png`

- 8 columns × 9 rows
- 192 × 208 pixels per cell
- Idle, movement, wave, jump, failure, waiting, electric blast, and inspection states
